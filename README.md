# snippets
各种编程语言的实用代码片段、技巧、常用函数集合。

目标：**可直接复制、或简单配置就能使用**的代码集合。

## 目录说明

语言 / 主题 | 说明
------------|------
[`python/`](./python)       | Python 常用函数、技巧、库封装
[`cpp/`](./cpp)             | C++ 小工具、STL 使用模式、性能小技巧
[`sql/`](./sql)             | 常用 SQL 查询模式、窗口函数、性能优化片段
[`shell/`](./shell)         | Bash / Zsh 常用脚本片段、文本处理、运维小工具
[`rust/`](./rust)           | Rust 常用模式、错误处理、命令行工具小示例
[`other/`](./other)         | 正则、Markdown、算法技巧等不适合单独分类的内容

## 使用方式

1. 直接浏览对应语言文件夹
2. 每个文件尽量保持**独立**，开头通常有简短说明和使用示例
3. 大部分文件可以直接运行
4. 有些文件通常需要包含在自己的项目中编译

## 命名约定

- 使用描述性的文件名
- 使用 `.md` 放纯文本
- 代码文件后缀保持语言原生
- 如果一个文件里有很多小函数，在文件顶部加目录

## 贡献

欢迎 PR！  
请尽量保持：

- 每个文件专注于一个主题
- 有清晰的注释和使用示例
- 代码可独立运行或测试
